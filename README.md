### Roadmap & Upcoming Features

- AVX-512 Support: Implementing Philox32x4-10x4 for next-generation CPU performance.

- SIMD Auto-Dispatching: The library will automatically detect CPU capabilities (AVX-512 / AVX2 / SSE) at runtime and switch to the optimal implementation.

- No manual configuration required. Maximum performance on every machine.

- Threefry Implementation: Adding a high-quality counter-based PRNG option for diverse simulation needs.

- High-Quality Seed Generator: Adding a robust, high-entropy seed generator for improved randomness initialization.
